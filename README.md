# 👨🏻‍💻 Customer Behavior Data Analysis Project

This project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI. It focuses on analyzing customer shopping behavior to generate meaningful business insights.

---

## 📌 Project Overview

The objective of this project is to analyze customer purchasing patterns and extract actionable insights that help businesses make data-driven decisions.

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* SQL (MySQL)
* Power BI
* Jupyter Notebook

---

## 📊 Project Workflow

### 1. Data Collection & Import

* Loaded dataset into Python environment

### 2. Data Cleaning & Preprocessing

* Handled missing values
* Converted data types
* Removed duplicates

### 3. Exploratory Data Analysis (EDA)

* Identified patterns and trends
* Visualized customer behavior

### 4. SQL Analysis

* Stored cleaned data in database
* Executed queries to extract business insights

### 5. Dashboard Creation

* Built interactive dashboard using Power BI

---

## 📈 Key Insights

* Identified high-value and top-spending customers
* Analyzed purchasing trends across product categories
* Observed repeat vs new customer behavior

---

## 🚀 How to Run This Project

1. Clone the repository:

```bash
git clone https://github.com/Rishana05/customer-behavior-analysis.git
```

2. Open Jupyter Notebook:

```bash
jupyter notebook
```

3. Run the notebook:

```
Customer_Shopping_Behavior_Analysis.ipynb
```

4. (Optional) Connect to SQL database and execute queries

5. Open Power BI file to view dashboard

---

## 📁 Project Files

* `Customer_Shopping_Behavior_Analysis.ipynb` → Python analysis
* `customer_behavior_sql_queries.sql` → SQL queries
* `customer_behavior_dashboard.pbix` → Power BI dashboard
